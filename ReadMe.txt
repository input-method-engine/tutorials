輸入法引擎製作教程（IME Developing Tutorials）
==================================================

該項目旨在提供一系列由淺入深的輸入法引擎製作教程。
This project aims to provide a series of step-by-step tutorials for developing
an input method engine (IME).

輸入法引擎僅爲輸入法提供具體實現，不提供用戶界面，從而適用於更多不同場合。
IMEs provide concrete implementations for input methods, without bindings to
any specific user interfaces, thus being more adaptable for different purposes.

目前該項目尚處於準備階段，不必過分期待。
Currently the work is still in preparation.  Don't starve for it.
